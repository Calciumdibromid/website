Calciumdibromid - Download



Calciumdibromid wurde von Informatikstudenten der TH Rosenheim für den Campus Burghausen programmiert.

Es ähnelt dem Programm "Beryllium10", welches von Studierenden der Universität Regensburg entwickelt wurde. Dieses wurde lange nicht mehr weiterentwickelt und wegen Änderungen in den Quellen ist es mittlerweile in seinen Funktionen eingeschränkt.


Da CaBr2 ein Open-Source-Projekt ist, kann man sich hier die aktuellste Version kostenlos downloaden:


  • Link zur Webapp CaBr2


Wer genauere Informationen zum Programm wissen möchte, kann sich den Code dazu auf GitHub anschauen.


Dort befinden sich auch genauere Informationen zur Lizenz. Diese besagt grob zusammengefasst, dass der Code verwendet und verändert werden darf, aber das Programm und die daraus entstehenden Programme Open-Source bleiben müssen.
